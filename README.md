# FLUTTERFrontend - TDD

## To run on local machine
   - **flutter run -d chrome**

## To run tests
   - Acceptance tests
     - run manually **test_driver/app.dart**
     - run manually **test_driver/app_test.dart** (in parallel) 
   - Widget tests
     - **flutter test**
   - Unit tests
     - **fluttter test**
